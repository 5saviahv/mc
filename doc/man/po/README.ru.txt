= Предварительные требования =

 * transifex-client (easy_install transifex-client)
 * po4a

== настройка transifex-client ==

в файл ~/.transifexrc внесиле строки:

[https://www.transifex.net]
hostname = https://www.transifex.net
password = <ваш пароль на Трансифексе>
token = 
username = <ваш логин на Трансифексе>

= Использование =

== Создание файла переводов ==

Создать pot-файл и закачать его на Трансифекс:
cd doc/man/po
make pot

== Создание переведённых man-страниц ==

Получить с Трансифекса переводы и сформировать man-страницы
cd doc/man/po
make man

= Структура =

l10n - переведённые маны
po - служебный каталог для переводов (для взаимодействия с Трансифексом)

po/mc      \
po/mcedit   - служебные каталоги для po-переводов MAN-страниц
po/mcview  /
